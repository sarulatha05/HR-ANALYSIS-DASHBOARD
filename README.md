HR Analytics Dashboard
Project Overview

This project focuses on analyzing employee attrition and workforce trends using SQL and Power BI. The dataset contains 1,470 employee records and was cleaned, transformed, and modeled to identify factors influencing employee turnover.

The dashboard provides insights into:

Employee attrition rate and active workforce
Attrition by department and job role
Salary band analysis
Age and gender demographics
Overtime impact on attrition
Job satisfaction, environment satisfaction, and work-life balance analysis
Workforce distribution across departments and roles
Tools Used
SQL (MySQL)
Power BI
DAX
Data Cleaning & Transformation
Key KPIs
Total Employees
Active Employees
Attrition Count
Attrition Rate
Average Salary
Average Employee Age
Key Insights
Research & Development recorded the highest employee attrition.
Overtime employees showed significantly higher turnover.
Lower salary bands experienced higher attrition levels.
Job satisfaction and work-life balance were strongly associated with employee retention.
Certain job roles contributed disproportionately to employee turnover.
